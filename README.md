## 🛠️ The Technical Checklist (Industry Standards)
Every project in this week's sprint was built with the following constraints:
- **PEP8 Standards:** Using `snake_case` and clean spacing for team-ready code.
- **Indentation:** 4-space standard to maintain logical hierarchy.
- **Data Integrity:** Strategic use of **Tuples (Immutable)** for constants and **Lists/Dicts (Mutable)** for dynamic data.
- **Environment Safety:** All development performed within a **Virtual Environment (venv)**.
