# Python Foundations 

A collection of core Python projects and logic exercises built as part of my 90-day AI Development roadmap. This repository tracks my progress from basic syntax to complex logic.

---

## Projects

### 1. Salary Hike Calculator
A robust tool designed to calculate new salary packages based on percentage increments. This was the inaugural project of this repository, focusing on clean financial logic.
* **Key Features:**
    * **Input Validation:** Handles non-numeric inputs gracefully using `try-except` blocks.
    * **Logic Protection:** Prevents negative salary or hike calculations.
    * **Professional Formatting:** Outputs results in standard currency format (e.g., `1,200,000.00`).

### 2. AI Startup Eligibility Bot
A screening tool that evaluates a candidate's readiness for AI roles based on a weighted scoring system.
* **Key Features:**
    * **Weighted Scoring:** Calculates eligibility based on specific skill importance (Python, ML, Backend).
    * **Interactive Input:** Uses a custom validation loop to ensure ratings stay between 1-5.
    * **Tier System:** Categorizes results into Tier 1, 2, or 3 based on final scores.

---

## Concepts Covered
* **Error Handling:** Extensive use of `try...except` for robust user experiences.
* **Data Validation:** Loop-based checks to ensure clean data entry.
* **Modular Code:** Using functions and `if __name__ == "__main__":` blocks for clean structure.
